## 🛠️ Technologies Used

- **Python** — Core programming language
- **Pandas / NumPy** — Data manipulation and analysis
- **Matplotlib / Seaborn** — Data visualization
- **Statistical Methods** — Descriptive and inferential statistics for retail insights

## 🚀 Getting Started

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn
```

### Running the Pipeline
Run each layer sequentially, starting from data generation through to final insight generation:

```bash
python "Data/data generation1.py"
python "cleaned data/Data Cleaning & Quality Control Layer.py"
python "data analysis/analytical engine.py"
python "data features/Feature Engineering.py"
python "data visualization/dashboeard visualization.py"
python "note books/bussiness reporting.py"
python "final result/finnal insight generation layer.py"
```
